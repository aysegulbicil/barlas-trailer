# Videolar klasörü

Bu klasöre koyduğunuz videolar, **Medya** sayfasının **Videolar**
sekmesinde otomatik olarak oynatıcı kartları halinde görünür.
Klasör boşken sekmede "Yakında" yer tutucu kartlar gösterilir.

## Nasıl eklenir?
1. Video dosyalarını (`.mp4`, `.webm`, `.ogg`, `.mov`) bu klasöre kopyalayın.
   (En geniş tarayıcı desteği için `.mp4` / H.264 önerilir.)
2. (İsteğe bağlı) Kapak görseli: video ile **aynı ada** sahip bir resim
   ekleyin → otomatik kapak (poster) olur.
   Örn. `tanitim.mp4` + `tanitim.jpg`
3. Sayfayı yenileyin.

## İpuçları
- Dosya adı, video başlığı olarak kullanılır
  (`uretim-sureci.mp4` → "Uretim Sureci").
- Sıralama için ad başına numara verin: `01-...`, `02-...`
