# Fotoğraflar klasörü

Bu klasöre koyduğunuz görseller, **Medya** sayfasının **Fotoğraflar**
sekmesinde otomatik olarak listelenir. Kod düzenlemeye gerek yoktur.

## Nasıl eklenir?
1. Görsel dosyalarını (`.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.avif`)
   doğrudan bu klasöre kopyalayın.
2. Sayfayı yenileyin — görseller alfabetik (doğal) sırada görünür.

## İpuçları
- Dosya adı, fotoğrafın alt metni/başlığı olarak kullanılır.
  Örn. `fabrika-uretim-01.jpg` → "Fabrika Uretim 01".
- Sıralamayı kontrol etmek için ad başına numara verin:
  `01-...`, `02-...`
- Web için optimize edilmiş (sıkıştırılmış) görseller kullanın;
  uzun kenar ~2000px genelde yeterlidir.
