# CSS Variables

Tüm renkler CSS değişkenleri üzerinden kullanılmalıdır.

```css
:root{
    --primary: #005BAA;
    --dark: #111827;
    --light: #F8FAFC;
    --white: #FFFFFF;
    --success: #22C55E;
    --danger: #EF4444;
}
```

## Kullanım Kuralı

Doğrudan HEX kodu kullanılmamalıdır.

Örnek:

```css
background: var(--primary);
color: var(--white);
border-color: var(--accent);
```

Tüm tema renkleri yalnızca CSS değişkenleri üzerinden yönetilmelidir.
