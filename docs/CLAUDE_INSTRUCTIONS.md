# CLAUDE_INSTRUCTIONS.md

Bu proje CodeIgniter 4 ile geliştirilmektedir.

Kurallar:

1. Mevcut mimariyi değiştirme.
2. Yeni framework önerme.
3. CodeIgniter 4 standartlarına uy.
4. Header ve Footer ortak yapıda olsun.
5. Layout sistemi kullan.
6. View içerisinde sabit metin yazma.
7. Tüm metinler language dosyalarından gelsin.
8. Responsive tasarım zorunludur.
9. SEO kurallarını uygula.
10. Bootstrap 5 kullanılabilir.
11. GSAP, AOS ve Swiper kullanılabilir.
12. Animasyon kodlarını ayrı dosyalarda tut.
13. CSS ve JS kodlarını modüler yaz.
14. Gereksiz kütüphane ekleme.
15. Kodları açıklamalı ve sürdürülebilir yaz.
